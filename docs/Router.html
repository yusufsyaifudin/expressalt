(static) configFunctionLoader(config)

Parse Config

Parsing configuration object and return array of middleware and alias name of this route. The config object contains `middleware` name and `as` name. Another object key will not counted as config.
Parameters:
Name Type Description
config object Array of Object of configuration
Source:
  • Router.js, line 604
Returns:
object with middleware key and as key. Middleware key contains array of callback function.
Examples
Router.configFunctionLoader({middleware: 'AuthMiddleware', as: 'route_name'});
See Router.loadMiddleware if you input a string as parameter.
Router.configFunctionLoader({as: 'route_name', middleware: (req, res, next) => {}});
Router.configFunctionLoader({middleware: [(req, res, next) => {}]});
Router.configFunctionLoader({middleware: ['MiddlewareName', (req, res, next) => {}]});

(static) routeRegister(verb, uri, param2, param3, additionalConfig)

Parse all configuration to function that ready to call and register to route.
Parameters:
Name Type Description
verb string http verb to be register, `GET`, `POST` or etc
uri string prefix of route
param2 object | function | string can be config (if object), callback function for expressjs or controller `classname.function_name`. See Router.callbackLoader
param3 function can be either `ControllerClass.FuncName` or callback function
additionalConfig object configuration like middleware or etc, See Router.configFunctionLoader * @example Router.routeRegister('GET', '/foo', {}, () => { }, { middleware: [], 'as': 'name' });
