Module: Router

Will return new Router class object. This avoid another class or function outside this class to call static method that is prohibited because static method like `Route.loadFunction`, `Router.callbackLoader` and `Router.routeRegister` are use to load function from controller. In brief, all static method in this class SHOULD NOT needed by user of this application.
